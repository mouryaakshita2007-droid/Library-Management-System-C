# 📚 Library Management System in C

A menu-driven Library Management System developed in C using file handling and modular programming.

## Features

- Admin and User Login
- Add Books
- Display Books
- Search Books
- Delete Books
- Add Members
- Issue Books
- Return Books
- Late Return Fine (₹5 per week after the due date)
- Book Not Found Validation
- File Handling using .dat files

## Technologies Used

- C Programming
- File Handling
- Structures
- Functions
- Modular Programming
- GCC Compiler (MinGW)

## Project Structure

```
main.c
books.c
books.h
members.c
members.h
utils.c
utils.h
books.dat
members.dat
```

## How to Compile

```bash
gcc main.c books.c members.c utils.c -o library.exe
```

## How to Run

```bash
library.exe
```
